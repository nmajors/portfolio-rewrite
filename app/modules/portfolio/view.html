Weather Watcher

Weather Watcher

Final Project

My final project for The Iron Yard. A fully responsive web app where each user can save addresses for their friends and family and keep track of weather conditions and severe weather situations for all of their saved locations in one place.

Promises. Promises everywhere. Geolocate the user then Geocode their location then get the weather for their current location then get their saved locations then get the weather for their saved locations then assign the map marker icon to each location according to the current weather for that location. Broken Promises = broken project.

  • HTML
  • CSS/SASS
  • AngularJS
  • Firebase
  • Angular Google Maps
  • OpenWeatherMap API
  • Bootstrap form styles
